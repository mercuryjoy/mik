  • 描述：POST /goods/orders

参数 数据类型 是否必传 含义 备注
item_id int 是 商品ID
amount int 是 数量 大于等于1
contact_name string 是 收货人
contact_phone string 是 联系号码
shipping_address string 是 收货地址 不能少于5个字符
